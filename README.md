# Integrisense Research

## Overview
This project investigates low-cost multimodal wrist-based stress detection using physiological signals and machine learning.

## Sensors (Embedded Deployment)
- GSR v1.2 (EDA)
- MAX30102 (PPG)
- MPU6050 (Accelerometer)
- MLX90614 (Temperature)
- ESP32

## Research Goals
- Replicate wrist-based WESAD methodology
- Implement feature-level fusion
- Evaluate cross-subject generalization (LOSO)
- Deploy optimized model to ESP32
